# DJ-PROPERTI

Platform untuk web iklan properti, meliputi, rumah, tanah, apartemen, dll. Platform ini dibuat dengan menggunakan python3 + django.
	
## Goals

Mimic Indonesian property listing site, such as olx, rumah123,
urbanindo, etc.
